Async API Server

A modern asynchronous API server built using FastAPI, one of the fastest Python web frameworks for building APIs. This project demonstrates how to create API routes, run a backend server, and handle HTTP requests using asynchronous programming.


---

Features

Fast asynchronous API server

Lightweight and high performance

Automatic API documentation

Beginner-friendly backend project

REST API development

Real-time API testing



---

Technologies Used

Python

FastAPI

Uvicorn

Installation

1. Clone the Repository

git clone https://github.com/your-username/async-api-server.git


---

2. Navigate to Project Folder

cd async-api-server


---

3. Create Virtual Environment (Optional)

python -m venv myenv

Activate environment:

Windows

myenv\Scripts\activate

Mac/Linux

source myenv/bin/activate


---

4. Install Required Libraries

pip install fastapi uvicorn

Or install using requirements file:

pip install -r requirements.txt
