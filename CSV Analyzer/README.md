CSV Analyzer

A beginner-friendly data analysis project built using Python and Pandas. This project reads CSV files, displays dataset information, and performs basic statistical analysis.


---

Features

Read CSV datasets

Display dataset preview

Generate statistical summary

Beginner-friendly data analysis project

Lightweight and easy to understand



---

Technologies Used

Python

Pandas

---

Installation

1. Clone the Repository

git clone https://github.com/your-username/csv-analyzer.git


---

2. Navigate to Project Folder

cd csv-analyzer


---

3. Create Virtual Environment (Optional)

python -m venv venv

Activate environment:

Windows

venv\Scripts\activate

Mac/Linux

source venv/bin/activate


---

4. Install Required Libraries

pip install pandas

Or install using requirements file:

pip install -r requirements.txt

---

How to Run the Project

Open terminal inside project folder and run:

python main.py
