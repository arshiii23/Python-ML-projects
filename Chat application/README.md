Chat Application

A real-time client-server chat application built using Python sockets. This project demonstrates the fundamentals of network programming, socket communication, and real-time messaging between two systems.


---

Features

Real-time messaging

Client-server architecture

Localhost communication

Simple and beginner-friendly

Lightweight terminal-based application

Built using Python socket programming



---

Technologies Used

Python

Socket Programming

TCP/IP Protocol

---

How It Works

The server waits for incoming client connections.

The client connects to the server using localhost and port number.

Both users can send and receive messages in real time.
